1. Problem Description

For my web development project, I will create a travel blog. A travel blog is simply a blog where the entire focus is on the area of travel. It is a type of website that is updated regularly in the form of posts and blogs. My travel blog will focus on four places that I have travelled to in Poland such as Krakow, Tatra Mountains, Warsaw and Masuria.

On the home page, I will create a menu bar on the top, where an end-user can find the links to register, log in, to contact the blog's creator, the project report and the four blogs. In the banner area, I will place an image with the flag of Poland, and in the middle, I will put a banner that welcomes the users to the blog. Afterwards, I will describe the main objective of my website which is showing to the potential user four areas in Poland that are worth seeing. I will place four posts that will advertise my four blogs of the tourist guides for each place. Moreover, on the right side of the page, I will add four boxes that will display a photo and information about the creator of the website, categories, popular posts and popular tags. Ultimately, at the bottom of this page, I will embed a Google map that will display four pinned places in Poland which I will talk about in my blog.

The second page will be my first blog about Krakow. It will have the same header and footer sections as the home page. 3. Technology Selection and Site Architecture

For my website, I will use the newest versions of HTML and CSS which are HTML5 and CSS3. HTML5 will enrich the semantic content of my website pages, new page structure elements such as main, section, header, footer and nav are also added. CSS 3 also brings several new styles that weren't previously seen in older versions. One great new class styling attribute is the rounded borders I could add to tables or div tags. The border-radius attribute defines the depth of roundness on a border. Finally, another common CSS 3 attribute is the shadowing effect. Shadows also give the buttons, text or div tags some depth and a 3-dimensional feel.

For my website, I planned a standard blog layout so that on the home page there will be a menu bar at the top, then the banner and then the posts. Then the other pages will be full of content, images, videos and so on.

I will make the website responsive with cross-browser compatibility so that the user will be able to view my blog on any device like a phone, tablet or laptop and on any browser.
